## 从尾到头打印链表:3rd_place_medal:

#### 题目描述

输入一个链表，从尾到头打印链表每个节点的值。

#### 代码:v:

思路：先将链表每个结点的值存入数组中，然后通过数组的reverse方法，即可从尾到头打印



```js
function printListFromTailToHead(head)
{
    // write code here
    var arr=[];
    while(head!=null){
        arr.push(head.val);
        head=head.next;
    }
    return arr.reverse()
}
```
